About toodope

Toodope is a completely free volunteer-run website that helps HLS students share outlines and professor evaluations with each other. The website only works because of people like you!

The website is open to anyone with a harvard.edu email account. The website asks you to login with this email account to verify that you have access to it. Unfortunately, if you don't have one of these, you won't be able to log in at this time.

If something looks broken to you, please email me at admin@toodope.org or open an issue. If you know how to code and want to help out, just reach out at admin@toodope.org. If you are curious, the code is open-sourced. The privacy policy is here.

Are you unable to sign in?
  • Make sure you're using your Harvard e-mail address
  • Enable JavaScript
  • If you use a privacy blocker, add "toodope.org" to your list of trusted websites
  • Use Google Chrome as your browser for accessing this website
  • Allow third-party cookies (the eye icon in address bar in Google Chrome should not have a line through it)
  • Make sure your address bar says http://www.toodope.org, not https://www.toodope.org. If you want to use https, you can go to https://toodope.herokuapp.com/.
  • E-mail admin@toodope.org for more help
